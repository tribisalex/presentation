Neural networks also have disadvantages:

  • The answer given by a neural network is always approximate – neural networks are not able to give accurate and unambiguous answers;
  • Inability to make decisions in several stages – a neural network cannot solve tasks that require sequential execution of several steps, it is able to solve the problem only "in one go";
  • Inability to solve computational problems – a neural network cannot solve problems that require sequential execution of several steps; it is able to solve the problem only "in one go";
  • Labor intensity and duration of training – it is required to train it on tens of millions of input data sets.

What is a neuron (processor)

A neuron is a computing unit that receives information, performs simple calculations on it and passes it on.

What is a synapse (signal)?

A synapse is a connection between neurons

Forward propagation

Forward propagation is the process of transmitting input values to a neural network and obtaining output data, which is called the predicted value
The formula for calculating the neural network layer is as follows:

f(x) = (inputs * weights) + bias

  • x is the result
  • inputs are input values
  • weights are weights (synapses)
  • bias is the deviation neuron (not always used)

The activation function

The activation function determines the output value of the neuron depending on the result of the weighted sum of inputs and the threshold value. In simple terms, the activation function is needed for data normalization

What are hyperparameters of the model?

Hyperparameters of a model are a configuration external to the model. Hyperparameters include: the number of neurons in the layer, the learning step, the number of iterations, the activation function, as well as the presence/absence of deviation neurons
